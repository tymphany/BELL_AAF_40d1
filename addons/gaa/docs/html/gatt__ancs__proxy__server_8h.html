◆ GattAncsProxyServerSendReadyCharNotification()

void GattAncsProxyServerSendReadyCharNotification ( uint16  cid)

Sends a notification for the ready characteristic to the remote client with this connection ID. Must only be called by the application in response to GATT_ANCS_PROXY_SERVER_READY_CHAR_UPDATE_IND. The application must determine if it is required to call this API or not (based on the ready characteristic configuration).

Parameters
